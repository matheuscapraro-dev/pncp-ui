# PNCP Search

Busca e análise de licitações do Portal Nacional de Contratações Públicas (PNCP).

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS 4
- Shadcn/ui
- Sonner (toasts)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
